# Template-v2

静的サイト構築のためのフロントエンド開発環境です。

## 導入

依存パッケージのインストール：

```bash
npm install
```

ローカルサーバーの起動：

```bash
npm run dev
```

## 開発用コマンド

```bash
npm run dev
```

ローカルサーバーを起動して、ファイルの変更監視を行います。

```bash
npm run build
```

ビルドしたファイルを `../htdocs` ディレクトリに出力します。
